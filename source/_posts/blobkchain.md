# 以太坊

Ethereum，数字货币代号**ETH**，是区块链2.0应用的代表。相比1.0：

| 区块链1.0（以比特币为代表）      | 区块链2.0（以太坊为代表）      |
| :------------------------------- | ------------------------------ |
| 非图灵完备：只能执行有限类型指定 | 图灵完备的                     |
| 不支持智能合约                   | 支持智能合约                   |
| 定位于支付网络，数字货币         | 定位于平台，可以衍生出各种应用 |
